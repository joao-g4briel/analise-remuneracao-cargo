# Análise de Remunerações - Cargos Comissionados (SIORG)

Análise exploratória das remunerações de cargos e funções comissionadas do governo federal brasileiro.

## Objetivo
Salário Mínimo, Máximo e Médio, entre tipos de cargos e principais insights sobre remunerações.

## Arquivos
- `br_me_siorg_remuneracao.csv` → Dados originais
- `remuneracao_cargo.xlsx` → Tabelas dinâmicas, gráficos e cálculos

## Principais Insights
- Remuneração média: R$ 397.954,02
- Maior valor: R$ 1.743.215,00
- Cargos com maior variação: FCE e CCE

## Como usar
1. Baixe o arquivo .xlsx
2. Explore as abas de Tabelas Dinâmicas e Gráficos

## Ferramentas utilizadas
- Microsoft Excel (Tabelas Dinâmicas + Gráficos)
